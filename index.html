Generalizable Diffusion Policy
with Transferable Affordance for Robotic Manipulation

Abstract

Diffusion-based policies have shown impressive performance in robotic manipulation tasks while struggling with out-of-domain distributions. Recent efforts attempted to enhance generalization to similar objects by incorporating additional knowledge, such as point cloud observations, 3D semantic fields, and SO(3)-equivariant networks. However, leveraging these priors is hard to generalize effectively to complex manipulation tasks for unseen objects and categories.

Our key insight is that leveraging affordances—manipulation priors that define "where" and "how" an agent interacts with an object—can substantially enhance generalization to entirely unseen object instances and categories. This paper introduces the Generalizable Diffusion Policy with Transferable Affordance (GeneralDP ), designed for generalizable manipulation across novel categories. Specifically, we model affordances as both the contact point and the post-contact trajectory, capturing the static and dynamic information essential for intricate tasks such as folding clothes. The transferable affordance from in-domain data to unseen objects is achieved by estimating a 6D transformation matrix through the use of foundational vision models and point cloud registration techniques. Additionally, we incorporate affordance guidance during diffusion sampling to ensure fine-grained control over the generation process. Experimental results, in both simulated and real-world environments, demonstrate that GeneralDP consistently outperforms previous diffusion-based methods, successfully generalizing to unseen instances and categories where others fail.

Overview

Given the target scene RGB-D image, GeneralDP retrieves a similar object in the affordance memory and transfers its affordance to the target object. Conditioned on 3D affordance and point cloud observation, GeneralDP utilizes the Diffusion Policy and adaptive affordance guidance for precise control.
